# 宝宝英语日课 — Parent-Child English Podcast

> 每天 15 分钟，和宝宝一起学地道英语。

## 项目概述

**产品定位**：帮助中国妈妈用地道英语和 2 岁宝宝互动，覆盖日常生活、绘本、游戏、情绪引导等场景。

**核心理念**：
- 克拉申 i+1：内容略高于当前水平，70-80% 可理解
- Bloom 掌握学习：听→理解→跟读→实践→反馈
- 不是课本英语，是"美国妈妈怎么说"

**四段式结构**（每集 15-18 分钟）：
1. **Scene 场景沉浸**（5 min）：一个亲子场景的英语对话
2. **Phrases 核心表达**（5 min）：5 个地道说法 + 费曼式讲解
3. **Shadow 跟读训练**（3 min）：8-10 句跟读 + 慢速重复
4. **Try It 实践引导**（2 min）：今天就能用的具体句子

---

## 技术栈

| 层 | 技术 |
|-----|------|
| 场景规划 LLM | 通义千问 qwen-plus |
| 脚本生成 LLM | 通义千问 qwen-max |
| 英文 TTS | Edge TTS (en-US-AriaNeural) |
| 中文 TTS | CosyVoice (longcheng) |
| 音频处理 | ffmpeg |
| 服务器 | 腾讯云 (复用递归大师基建) |
| 语言 | Python 3.9+ |

---

## 场景主题库

| 类别 | 场景数 | 示例 |
|------|--------|------|
| 日常作息 | 10 | 起床、穿衣、刷牙、吃饭、洗澡、睡前 |
| 绘本共读 | 5 | 一起读、提问、描述画面、复述、唱儿歌 |
| 户外活动 | 5 | 游乐场、散步、超市、公园、车上 |
| 游戏互动 | 7 | 积木、画画、橡皮泥、躲猫猫、过家家 |
| 情绪引导 | 6 | 安抚、表扬、边界、分享、道歉、发脾气 |
| 认知启蒙 | 6 | 颜色、数字、动物、身体部位、天气、食物 |
| **合计** | **39** | 约 6 周不重复内容 |

---

## Pipeline 架构

```
场景主题库 + learner_state.json
        ↓
   [1] 选择今日场景 (qwen-plus)
        ↓
   [2] 生成四段式脚本 (qwen-max)
        ↓
   [3] 质量检查
        ↓
   [4] 双语 TTS 合成
       英文: Edge TTS (AriaNeural)
       中文: CosyVoice (longcheng)
        ↓
   [5] 音频组装 (ffmpeg)
       含 2 秒跟读空白
        ↓
   [6] 输出 MP3 + 文字版脚本
```
