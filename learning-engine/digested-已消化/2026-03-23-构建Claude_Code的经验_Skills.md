# Skills 文章消化笔记
日期：2026-03-23

## 我理解了什么
- Skill 的本质不是"流程自动化"，而是"经验化扩展"（extension points）
- 经验分三层：该怎么做 / 不该怎么做（踩坑点）/ 做了会怎样（验证标准）
- 踩坑点不是一次性写完的，是在使用中逐步积累的——这是我之前缺的

## 我没理解什么
- 渐进式披露（progressive disclosure）、按需钩子（on-demand hooks）等概念能跟上，但不知道具体怎么用
- 原因：我还没有从零完整构建过一个 Skill，缺乏动手经验去对应这些概念
