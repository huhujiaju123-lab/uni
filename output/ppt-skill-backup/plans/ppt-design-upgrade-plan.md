### 6. 页面布局与间距

```
┌─────────────────────────────────────────────────┐
│ [鹿头Logo]  大标题(40pt Bold)                    │ ← top=0.35"
│              副标题说明(16pt, #666)               │ ← top=1.5"
│                                                   │
│  ┌─ 结论框 ────────────────────────────────────┐ │ ← top=2.0"
│  │ 结论：xxxx (28pt Bold, #0F3460)              │ │
│  │ 1. 要点... (20pt, #333)                      │ │
│  │ 2. 要点...                                   │ │
│  └──────────────────────────────────────────────┘ │
│                                                   │
│  章节标题 (24pt Bold, #0F3460)                    │ ← 结论框下 0.3"
│  ┌─ 表格 ──────────────────────────────────────┐ │ ← 标题下 0.1"
│  │ 表头 (深蓝底白字)                             │ │
│  │ 数据行 (斑马纹)                               │ │
│  └──────────────────────────────────────────────┘ │
│                                                   │
│  章节标题2                                        │ ← 表格下 0.3"
│  ┌─ 表格2 / 图表 ─────────────────────────────┐ │
│  │                                              │ │
│  └──────────────────────────────────────────────┘ │
│                                                   │
│  *脚注 (14pt, #999)                  [luckin logo]│ ← 底部

```

**关键间距规则（避免重叠）**：
- 结论框与章节标题间距：0.3"（274320 EMU）
- 章节标题与表格/图表间距：0.1"（91440 EMU）
- 表格与下一章节间距：0.3"（274320 EMU）
- 图表与下一元素间距：0.3"
- 内容区左边距：1.5"（固定）
- 内容区宽度：23"（固定）

## 具体改动

### generate_ppt.py 改动点：

1. **字体全局替换**：`FONT_TITLE` 和 `FONT_BODY` 统一改为 `STKaiti`
2. **字号调整**：所有元素按新字号体系
3. **行间距**：所有段落设置 `line_spacing` 和 `space_before/space_after`
4. **结论框重写**：浅蓝底 + 左侧深蓝竖条（或深蓝标题栏方案）
5. **表格升级**：深蓝表头 + 斑马纹交替行 + 更紧凑的单元格内边距
6. **图表增强**：
   - 统一图例位置为顶部
   - 添加数据标签
   - 柱状图设置间隙比
   - 折线图设标记点样式
   - 饼图添加类别+百分比标签
   - 条件网格线
7. **间距系统**：各元素间用固定间距常量，避免重叠
8. **封面字号调整**：60pt → 54pt

## 验证方案

改完后重新生成测试 PPT（与之前相同内容），打开对比：
1. 结论框是否有质感
2. 表格是否有层次
3. 图表是否有数据标签和图例
4. 各元素间距是否均匀、无重叠
5. 字体是否全部为楷体-简
