## 结论框

- 背景：#EDF2F9（极浅蓝灰）
- 左边线：4px 深蓝竖条 #222773
- 无外边框
- 标题：28pt Bold #0F3460
- 要点：20pt Regular #333333，行间距 1.5 倍

## 表格

- 表头：#222773 深蓝底 + 白色文字 18pt Bold
- 奇数行：白色
- 偶数行：#F5F7FA
- 边框：1px #E0E0E0
- 内容：16pt Regular #333333

## 图表规则

| 属性 | 规则 |
|------|------|
| 图例位置 | 顶部(TOP)，不占图表面积 |
| 图例字体 | STKaiti 14pt |
| 坐标轴字体 | STKaiti 12pt |
| 坐标轴线 | 浅灰色 #CCCCCC |
