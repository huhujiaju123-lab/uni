# Life OS 使用说明书

> 写给从零开始的你。不需要懂编程，跟着步骤来就行。

---

## 这是什么？

Life OS 是一套跑在 Claude Code（命令行 AI 助手）里的"精力管理系统"。

它不是待办清单，不是效率工具。它做的事情很简单：

- 早上帮你想清楚今天最重要的一件事
- 晚上帮你回顾今天的精力状态，打个分，记下来
- 积累够数据后，帮你发现自己的行为模式

三个角色各司其职：

| 角色 | 干什么 | 什么时候用 |
|------|--------|------------|
| **Rem** (早晨伙伴) | 协商今天的 1 个焦点 | 早上说"早上好" |
| **Alfred** (管家) | 晚间精力复盘，打分归档 | 晚上说"复盘" |
| **Tyrion** (军师) | 用数据帮你看见行为模式 | 积累 4 周数据后，说"周策略" |

---

## 第一步：安装 Claude Code

Claude Code 是 Anthropic 出的命令行 AI 工具。打开终端（Terminal），运行：

```bash
npm install -g @anthropic-ai/claude-code
```

> 前提：电脑上需要有 Node.js（18+）。没有的话先装：https://nodejs.org
>
> 装好后在终端输入 `claude` 回车，按提示登录 Anthropic 账号就行。

验证安装成功：

```bash
claude --version
```

能看到版本号就 OK 了。

---

## 第二步：安装 Life OS 技能包

需要把三个技能文件放到 Claude 的技能目录下。让安装过的人（比如宵霄）把以下三个文件夹发给你：

```
~/.claude/skills/life-os-alfred/    # 管家（晚间复盘）
~/.claude/skills/life-os-rem/       # 早晨伙伴
~/.claude/skills/life-os-tyrion/    # 军师（周策略）
```

把它们放到你自己电脑的 `~/.claude/skills/` 目录下。如果这个目录不存在，先创建：

```bash
mkdir -p ~/.claude/skills
```

然后把三个文件夹复制进去就行。

---

## 第三步：初始化你的 Life OS

在终端里运行以下命令，创建目录结构：

```bash
mkdir -p ~/.claude/life-os/state
mkdir -p ~/.claude/life-os/archive/daily
mkdir -p ~/.claude/life-os/evolution
mkdir -p ~/.claude/life-os/weekly
```

然后创建你的个人画像文件：

```bash
touch ~/.claude/life-os/profile.md
```

用任意文本编辑器打开 `~/.claude/life-os/profile.md`，填写你的基本信息。格式参考：

```markdown
# Profile | 你的名字

## 基本信息

- 年龄：XX 岁
- 工作：你做什么的
- 坐标：你在哪

## 精力模式（慢慢填，先写你知道的）

- **充电方式**：什么事情让你精力充沛？（比如：运动、独处、读书……）
- **耗电信号**：什么事情让你疲惫？（比如：社交应酬、被打断、目标太多……）

## 沟通偏好

- 你喜欢 AI 怎么跟你说话？（简洁/温和/直接？）
- 有什么雷区？（比如：不要说教、不要鸡汤……）
```

不用写很多，几行就够。以后随时可以补充。

## 第四步：开始使用

在终端输入 `claude` 启动对话，然后：

### 早上：和 Rem 开始一天

直接说：

> 早上好

或者：

> /rem

Rem 会看你昨天的精力状态，问你今天想把精力放在什么上面，帮你定下 **1 个焦点**。整个过程 2-3 轮对话，一两分钟搞定。

### 白天：随时聊

想说什么就说什么。做了个什么事、有个什么想法、累了、开心了，都可以跟 Claude 说。它会帮你记在当天的状态文件里。

### 晚上：和 Alfred 复盘

说：

> 复盘

或者：

> /alfred

Alfred 会问你今天感觉怎么样、精力还剩多少，然后请你打个分（1-10），给今天贴几个标签（比如"深度工作""运动充电""社交消耗"），帮你把记录归档。整个过程 3-5 轮对话。

### 积累数据后：Tyrion 周策略

连续用了 4 周以后（精力日志有 28 条以上），说：

> 周策略

或者：

> /tyrion

Tyrion 会把你这一周的精力数据拉出来，告诉你哪些天状态好、哪些天状态差，发现规律，然后用提问的方式引导你思考。它不会给你建议，只会帮你"看见"。

## 核心原则

这套系统有几条底线，所有角色都遵守：

1. **不评判**。不会说"今天效率不高""你应该更自律"。永远不会。
2. **不列待办**。不会说"明天你要做 X、Y、Z"。焦点最多 1-2 个。
3. **关心能量流向，不关心勾了几个待办**。今天精力花在哪了、感觉怎么样，比完成了几件事重要。
4. **文件记忆 > 对话记忆**。所有记录都写进文件。关掉终端窗口、换台电脑，数据还在。
5. **跳过几天完全没关系**。忘了用、忙了、不想用，随时回来接着用，没有任何愧疚成本。系统会自然接续。
6. **环境设计 > 意志力**。不靠"逼自己坚持"，靠系统自然运转。

## 目录结构速查

```
~/.claude/life-os/
├── profile.md                    # 你的个人画像
├── state/
│   ├── today.md                  # 今天的状态（每天自动刷新）
│   ├── focus-queue.md            # 今日焦点（Rem 写入）
│   └── energy-log.jsonl          # 精力日志（Alfred 每天追加一行）
├── archive/
│   └── daily/
│       ├── 2026-03-14.md         # 每天的归档（Alfred 晚上归档）
│       ├── 2026-03-15.md
│       └── ...
├── evolution/
│   ├── patterns.md               # 行为模式（Tyrion 观察后更新）
│   └── milestones.md             # 里程碑
└── weekly/
    └── 2026-W12.md               # 周报（Tyrion 生成）
```

简单来说：
- `state/` = 今天正在发生的事
- `archive/` = 历史记录
- `evolution/` = 长期观察到的模式和成长
- `weekly/` = 每周总结

## 常见问题

**Q：我不会用终端怎么办？**
打开"终端"App（Mac 在启动台搜"终端"，Windows 搜"PowerShell"），输入 `claude` 回车，就进入对话界面了。跟聊天一样，打字就行。

**Q：忘了用好几天怎么办？**
什么都不用做。下次打开说"早上好"或"复盘"，系统会自动接上。不会有人问你"这几天去哪了"。

**Q：数据存在哪？安全吗？**
全部存在你自己电脑的 `~/.claude/life-os/` 目录下，纯文本文件。不上传任何服务器，不同步云端（除非你自己设置）。

**Q：能在手机上用吗？**
目前 Claude Code 只支持电脑终端。手机上暂时不行。

**Q：profile.md 不知道写什么？**
先写名字和一句话就行。用着用着你会知道该补充什么。
