## 人群包总数统计

### 按组合方式

每个基础组（G1-G3, C1-C3）可产生的行为标签组合：

```
2^3 = 8种组合
├─ 无标签（常规用户）        1种
├─ 单标签                     3种
├─ 双标签                     3种
└─ 三标签                     1种
```

**总人群包数：** 6个基础组 × 8种组合 = **48个细分人群包**

### 人群包层级

```
第一层：涨价 vs 不涨价（2组）
第二层：生命周期（3组）→ 共6个基础组
第三层：行为标签组合（8种）→ 共48个细分包
```
