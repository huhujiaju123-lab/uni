## 一、核心结论表

| 维度 | 国内经验/建议 | Lucky US 现状 | 差距与行动 |
|------|-----------|-------------|--------|
| **次月留存基线** | 国内现在 ~20%出头（但国内已运营多年，新客质量越来越差是正常的） | 15-18%（部分门店） | 考虑门店生命周期后，差距没有想象大 |
| **马来对标** | 马来开业 6-9 个月时做到近 30%，但当时已 40+ 家店 | US 仅 12 家店 | **门店密度是核心变量**，不能简单对标 |
| **新客券包** | 国内包了很多张 1.5 折券（约 ¥9.9），力度极大 | $1.99 / $2.99 / $3.99 / 5折，面额跨度大 | $2.99 和 $3.99 券基本废券，**需重新设计券包** |
| **券包拆分** | 按时间拆开：0-7天一张 / 7-14天一张，覆盖 30 天 | 仅首单券（$1.99 次日用），后续无系统券 | **至少 2-3 张券，按时间窗口拆开** |
| **触达频率** | 新用户最多一周一次，不同触点加起来只触达一次 | 短信 + Push（券到期提醒） | 频控很重要，**避免过度打扰** |
| **新人专区** | 注册后直接跳转新人菜单页（简化品类 + 首位价展示 + 塞券） | 无专区，注册后进普通首页 | **产品功能缺失，需提需求** |
| **单品引导** | 放爆品（生椰），用低价单品券（¥0.99 生椰浆）引导尝试 | 品类上没有特别强的引导 | **用低价单品券引导高复购品类** |
| **站外滚动培养** | 首单后 D3/D7/D14/D30 每天滚动检索未二单用户，塞券触达 | 目前是 0→1 阶段，还没做 | **最核心的建议：建立滚动培养机制** |
| **门店物料** | A字架/易拉宝放门口，覆盖动线人群，周期性领券 | 有物料但未充分利用 | 店少更要铺线下物料 |
| **裂变** | 裂变主要拉首单，对复购没什么提升 | 分享有礼拉新占比 ~5% | 裂变归裂变，复购归复购，**分开看** |
| **实验方法** | 所有策略都留对照组；最优实验组合并为新对照组，持续迭代 | 有实验但策略较分散 | **需要整合成基础策略，再叠加测试** |

## 二、三条核心路线（国内同事总结）

| # | 路线 | 具体动作 | 优先级 |
|---|------|--------|------|
| 1 | **新人券包加深** | 拉长券包时间窗口（0-7天 / 7-14天），去掉没用的高面额券（$3.99/$5折），换成多张低价券 | P0 |
| 2 | **站内营销工具** | 首单后弹窗领券、新人专区、付费券包、任务体系，让新客多看到你 | P0 |
| 3 | **站外滚动触达** | D3/D7/D14/D30 滚动客群，塞券 + 触达（Push/短信），一周最多一次 | P0 |

## 三、券策略详解

| 问题 | 国内经验 | 对 US 的启示 |
|------|--------|-----------|
| **现有券面额太大** | 单杯 $3 左右，发 $2.99/$3.99 券基本等于免费，没有价格感知差异 | 把 $2.99+$3.99+5折 换成 **多张 $1.99**，或者用百分比折扣 |
| **券的时间窗口** | 国内拆成两个时段：0-7天一张 + 7-14天一张 | 覆盖 30 天，至少 2-3 张券按时间拆开 |
| **券到期次日才能用** | 首单券次日生效是对的，防止当天用完没有浮动 | 保持现有逻辑 |
| **塞券 vs 提醒** | 塞券的核心不是核销，而是让用户知道"我还有券"→提高回访率 | 文案要写"送你一张券"而非"提醒你还有券"，**获得感**不同 |
| **塞券力度** | 必须低于大盘价 70-80%，新客要给获得感 | 塞 $1.99 或低于大盘的专属券 |

## 四、新人专区（产品功能）

| 要素 | 国内做法 | US 现状 |
|------|--------|-------|
| **注册跳转** | 新用户注册完直接跳转到新人专区（非普通首页） | 注册后进普通首页 |
| **专区内容** | 简化菜单 + 首位价展示 + 新人钱包里的券全部展出 | 无专区 |
| **菜单简化** | 只展示新客爱喝的 TOP 品，品名直白（"爱喝美式""爱喝冰"） | 完整菜单，新客找品难 |
| **新品+IP** | 每两周换 IP，新品/周边放进新人专区 | 无 |
| **AB 测试** | 新人菜单方案（自己配 vs 大盘运营配），自己配的更好 | 不能分流新客（产品限制），只能全局切换 |

**US 产品限制**：新客不能分流、没有专区功能、券包只能配一个（全局）。建议先争取前后对比测试。

## 五、滚动培养机制

| 节点 | 动作 | 说明 |
|------|------|------|
| **首单后 D3** | 检索未下二单的用户 → 塞券 + 触达 | 客群每天滚动更新 |
| **首单后 D7** | 同上，换不同券/文案 | 覆盖第一周没行动的人 |
| **首单后 D14** | 同上 | 二次触达 |
| **首单后 D30** | 最后一次触达 | 最多覆盖到 30 天 |
| **触达方式** | Push / 短信 / APP 内弹窗 | 每人每周最多 1 次触达 |
| **触达频控** | 如果今天 Push 了，短信就不发了 | 避免过度打扰 → 取关/删 c/退订 |
| **文案策略** | "送你一张券，快到期了" 比 "提醒你还有券" 好 | 获得感 > 提醒感 |
| **不需要管的人** | 首单后已经自然活跃（1→2，2→3 转化了），不用触达 | 识别流失趋势的人才需要干预 |

## 六、门店维度分析

| 发现 | 详情 |
|------|------|
| **开店首月新客留存最高** | 30-40% 次月复购，因为开业吸引的都是咖啡忠实用户 |
| **第 3-4 个月开始下降** | 周边该成新客的都成了，剩下的是路过/尝鲜客，质量下降 |
| **之后趋于稳定** | 不会一直低，后面会起一点，然后基本 log 曲线 |
| **店少 = 门店分析更有意义** | 12 家店可以逐店看留存，找共性/异常，策略匹配 |
| **机场店排除** | 纯流量型，不指望复购，不纳入分析 |
| **关注门店类型** | 商场 / 写字楼 / 学校三类，机场不看 |
| **学校店** | 体量最大（4-5 家），9 月开学季新客暴增，学生复购高 |
| **学生优惠** | US 已有学生认证 → 每月 2 张 5 折券（老客复购上用） |

## 七、单品策略

| 建议 | 说明 |
|------|------|
| **推爆品** | 国内首推生椰拿铁（市占最高、成本低、大部分人不烦） |
| **用低价单品券** | $0.99 生椰浆，用低价让新客尝试 → 种草 → 自然复购 |
| **降低踩雷概率** | 用品类引导降低首单选错品的风险（马来瑞纳冰教训：新客占比可以但复购差） |
| **看二单品分布** | 分析复购用户第二单喝什么，用这个品来引导首单/二单 |
| **US 现状** | 生椰占比 20 多，没有特别突出的爆品差异，品类引导不够强 |

## 八、其他策略

| 策略 | 国内做法 | US 适用性 |
|------|--------|---------|
| **成长任务** | 新客完成任务获奖励（在新客券基础上叠加），效果最好 | 可以做，但新客量少 |
| **付费券包/次卡** | 随单购入口推次卡，cover 本单消费 | 新客一般不会买，开卡率低，暂不折腾 |
| **线下物料** | A字架/易拉宝放门口，每周来领券，拉新+复购 | **店少更适合做**，铺物料覆盖动线 |
| **竞对门店券** | 新加坡在竞争门店(星巴克)旁放码，每天领 $3.99 | 旁边有星巴克的店可以试 |
| **假期活动** | 抽奖/IP 周边/节假日促销，拉回老客也带新客 | 开学季（9月）/ 节日可做 |
| **品控频控** | 国内品控严格（1V1 三天内不超频），过度触达 → 退订率高 | 必须控频 |
| **裂变物料** | 邀 4 人得杯子（实物奖励），国内电商发货 | US 物流贵，改为到店拿或给兑换券 |

| 原则 | 说明 |
|------|------|

## 十、关键认知

1. **门店密度是留存的底层变量** — 用户喝完首杯后覆盖不到就不会再来，这跟策略无关
2. **国内 20% 的留存不算高** — 因为运营多年，新客质量自然下降，早期也是 30%+
3. **券的核心不是核销，是回访** — 塞券 → 用户知道有券 → 回访率提升，即使不核销也值
4. **新客策略 = 基础整合，不是零散实验** — 先搭基础框架（券包+触达+专区），再叠加测试
5. **裂变不提留存** — 裂变是拉新量的工具，复购靠券包+触达+品类引导
6. **养到 4 单就稳了** — 国内数据：4 单以内的用户复购不稳定，4 单以上相对稳定
