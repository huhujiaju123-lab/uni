# OpenClaw 龙虾大赛 — 递归大师

## 项目概述

- **项目名称**：递归大师
- **主题**：让 AI 在工作中活起来 — 记忆力·元认知·执行稳定性
- **团队**：美国增长运营团队（3人）
- **比赛时间**：2026-03-18

## 核心理念

不是训练虾，而是跟虾共事。每次对话就是一次迭代。

三个维度：
1. **记忆力** — 犯过的错不再犯，经验能沉淀、能传播
2. **元认知** — 能意识到自己在重复，主动优化工作方式
3. **执行稳定性** — 不编答案，先对齐口径

## Q&A 杀手锏

如果被问「下一步打算做什么」：

> 虾积累了足够模式后，不只沉淀成技能，还能自己判断——这件事应该做成被动技能（Skill，说一句话触发）还是主动代理（Agent，自动定时执行）。从「你叫它干」变成「它自己知道该干」。
