# 佑安质量标准 — 内容筛选清单

> 基于佑安大师反馈总结的内容质量标准。
> 每卦内容生成后，必须逐项审查，全部通过后方可产出。

---

## 一、字义深度 (Character Depth)

每个关键字必须达到以下层次：

| 层次 | 要求 | 佑安示范 |
|------|------|----------|
| L1 本义 | 说文解字/甲骨文的**具体引文**，不是泛泛说"本义为X" | "屯，难也。象草木之初生。屯然而难。从屮贯一。一，地也。尾曲。" |
| L2 多义 | 同一字的多重含义、不同读音 | "屯"zhūn=艰难，tún=聚集/驻守/村落 |
| L3 取象 | 为什么作者**选这个字**而非其他字，字与卦象的对应 | "磐桓"取震象（雷为木），戒动故说"磐"而非"履" |
| L4 修辞 | 字与字之间的呼应、对仗、用典 | "匪寇婚媾"是古文语法倒装，先以为是匪寇，实则是求婚 |

**不合格表现**: 仅停留在 L1（说个本义就跳到引申），缺少 L3/L4 层。

---

## 二、取象精度 (Imagery Precision)

每个爻辞的解读必须：

- [ ] 追溯到**具体的八卦取象**（震为雷/为动/为长子/为龙/为马；坎为水/为险/为血/为云）
- [ ] 说明**为什么这个意象出现在这个位置**（爻位+卦体+互卦）
- [ ] 大象传与卦辞取象的**异同分析**（如卦辞说"水雷屯"，象传说"云雷屯"——为什么？）
- [ ] 乘承比应关系的**完整说明**（阳乘阴、阴承阳、比邻、相应）

**佑安示范**: "乘马班如"——"乘"是处于的意思，马取坎象（高脊马，险中之马），"班"同"盘"取盘旋不前之象。不是简单说"骑马盘旋"。

**不合格表现**: 跳过取象直接给白话翻译，或仅说"象征困难"而不解释为什么。

---

## 三、彖传/象传解读

- [ ] 不仅翻译，还要解释**孔子为什么这样取象**
- [ ] 彖传的**结构拆解**（通常是：释卦名 → 释卦体 → 释卦辞）
- [ ] 象传中孔子取象与卦辞原文的**差异分析**
- [ ] 每个字的取象来源标注

**佑安示范**: 象传说"云雷屯"而非"水雷屯"——"云者，物之将形；雷者，阳之初动。合而言之，物之初稚，气之将形，此屯之取象也。"

---

## 四、征引厚度 (Citation Depth)

历史征引不是一句话概括，而是：

- [ ] **完整讲述故事**（人物、背景、经过、结果）
- [ ] **直接引用原文**（如《明史》原文、诗词全文）
- [ ] **回扣爻义**——故事如何精确对应爻辞的每一个关键词
- [ ] **多文明互参**（儒/道/佛/诸子/史记/诗经至少两种）

**佑安示范**: 六四爻——不仅讲陈平投奔刘邦的故事，还完整引用曹操《短歌行》全文，再用南怀瑾的话收束。

**不合格表现**: "如周武王伐纣"一句带过，无细节、无原文引用。

---

## 五、映射锐度 (Mapping Sharpness)

现代映射不是泛泛的"如果你..."，而是：

- [ ] 具体到**可辨识的社会现象/案例**（如传销骗局、蚁力神事件）
- [ ] 引发读者的**深层价值观思考**（如佑安问："如果你是那个女人，你会怎么做？"）
- [ ] 有**逆向思考**——不仅说"应该怎么做"，还问"如果不这样做会怎样"
- [ ] 与**当下社会痛点**关联（如赚快钱、利己主义、信息茧房）

**不合格表现**: "你刚入职一家新公司"——太模板化，缺乏灵魂。

---

## 六、层次递进 (Layer Progression)

每个爻的解读必须体现四层递进：

```
字义层 → 象义层 → 人事层 → 哲理层
（说文训诂）→（八卦取象）→（历史/现实对照）→（修身处世智慧）
```

每一层有**独立的认知价值**，不能跳层。

**佑安示范（六三爻）**:
1. 字义层："鹿通禄取震象"、"虞"是猎人/向导、"林"取艮象
2. 象义层：三爻在震卦之末，动力衰减，故贪求外物必然求不得
3. 人事层：传销/诈骗的"即鹿无虞"模式（蚁力神、网络赌博）
4. 哲理层：孔子"君子爱财取之有道"→认知之外的钱别碰

---

## 七、AI 独创标准

在达到佑安质量基线后，AI 独创部分需要：

- [ ] **跨卦系统分析**: 本卦关键意象在64卦中的分布和演变
- [ ] **关系网络**: 综卦/错卦/互卦/序卦的深层含义（不是简单列举）
- [ ] **现代框架映射**: 心理学/决策科学/系统思维等学科视角
- [ ] 每条AI分析都有明确的**推理过程**，不是断言

---

## 评分方式

每项 1-5 分，总分 35 分。**≥28 分方可产出**。

| 维度 | 1分（不合格） | 3分（基本） | 5分（佑安级） |
|------|-------------|-----------|-------------|
| 字义深度 | 仅 L1 | L1+L2 | L1-L4 全覆盖 |
| 取象精度 | 无取象 | 部分取象 | 每词追溯卦象 |
| 彖象解读 | 仅翻译 | 翻译+简析 | 异同分析+取象溯源 |
| 征引厚度 | 一句话 | 简述故事 | 完整故事+原文引用 |
| 映射锐度 | 模板化 | 有具体场景 | 社会痛点+价值观拷问 |
| 层次递进 | 扁平 | 两层 | 四层递进 |
| AI 独创 | 无 | 有但浅 | 有推理过程的洞察 |
