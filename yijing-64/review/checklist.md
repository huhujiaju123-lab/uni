# 易经64卦 内容评审清单

## 一、专家学者视角

### 经典引用准确性
- [ ] 卦辞/爻辞引用是否与原文完全一致？
- [ ] 彖传/象传/文言传引用是否准确？
- [ ] 序卦传/杂卦传/说卦传引用是否有据？
- [ ] 系辞传引用是否准确？

### 训诂考据
- [ ] 字源追溯是否有《说文》或甲骨文/金文依据？
- [ ] 引申义是否合理，有没有过度解读？
- [ ] 佑安的训诂引用是否标注了出处？

### 爻位分析
- [ ] 阴阳/当位/不当位判断是否正确？
- [ ] 中正/应与关系是否分析准确？
- [ ] 重刚/重柔等特殊位置是否标注？
- [ ] 整体爻位逻辑是否自洽？

### 结构 vs 叙事 校验（防止用叙事逻辑覆盖结构逻辑）
- [ ] 九五是否被描述为「君位」而非「未来的/最终的」君位？
- [ ] 六爻是否被当作同时存在的结构模型，而非时间线？
- [ ] 初爻是否被描述为「初位」而非叙事「起点」？
- [ ] 上爻是否被描述为「极位」而非叙事「终点」？
- [ ] 当位/中正/应与 等结构属性是否未被加上时间副词？
- [ ] 爻与爻之间的关系用的是「位置/角色/处境」而非「旅程/到达/出发」？

### 历史征引
- [ ] 每条征引是否可查证？（出处+具体内容）
- [ ] 是否有虚构的历史事件或人物？
- [ ] 征引与爻义的对应是否恰当？

### 来源标注
- [ ] 所有内容是否标注了来源（经典/佑安/AI）？
- [ ] 来源标注是否准确？
- [ ] AI推演部分是否标注了不确定性？

## 二、普通读者视角

### 可读性
- [ ] 每段文字是否通顺易懂？（大学学历标准）

- [ ] 阅读节奏是否舒适？哪里太长/太跳？

- [ ] 现代映射能否代入读者自身生活？

- [ ] 征引故事是否有感染力？
- [ ] 映射场景是否真实可感？
- [ ] 整体基调是否一致（不是冷冰冰的百科）？
