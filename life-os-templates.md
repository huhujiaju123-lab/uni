# LifeOS 模板

## A. 每日精力记录模板（Alfred）

```json
{"date":"YYYY-MM-DD","score":7,"tags":["深度工作","运动充电"],"note":"今天完成X，体感中上，晚间略疲惫","timestamp":"YYYY-MM-DDTHH:MM:SS"}
```

字段约束：
1. `score`：1-10
2. `tags`：1-3 个
3. `note`：一句话，聚焦精力体验

## B. 周策略输出模板（Tyrion）

1. 周期：`YYYY-MM-DD ~ YYYY-MM-DD`
2. 数据概览：
   - 日志条数：X
   - 均值：X.X
   - 最高 / 最低：X / X
3. 拉高因素（Top 3）
4. 拉低因素（Top 3）
5. 本周变化（前后对比 5-7 维）
6. 下周单一策略问题（1个）

## C. 每周验收清单

1. 本周是否有 >= 5 条有效日志
2. 是否包含至少 1 次高分日与1次低分日分析
3. 是否沉淀一个“可复用规律”
4. 是否完成一次 learning-engine 同步
